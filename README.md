## projects

- **c++ systems lab**  
  structured repository exploring low-level systems concepts such as memory, processes, and disk i/o through hands-on experiments.

- **miniDB-KV (storage engine)**  
  a persistent key-value store built from scratch using file descriptors, binary serialization, and manual offset management for fast disk access.

- **linux memory monitor**  
  a lightweight tool that reads process memory usage directly from /proc, exposing real rss metrics and demonstrating kernel-level introspection.

- **mini shell (fork + exec)**  
  implementation of a basic unix shell to understand process creation, process duplication, and program execution.

- **automation & scraping (python)**  
  scripts using selenium and web scraping techniques for automation and data extraction.
